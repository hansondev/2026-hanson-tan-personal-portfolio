# Hanson Dev

> Category: Brands

> Surface: web

*Expert software engineering for developers and businesses who demand precision.*

Hanson Dev is a personal brand for a software engineer specializing in high-precision development services. The visual identity conveys technical excellence, stability, and professional credibility through a deep blue palette and clean typography.

## Color Palette

| Role | Name | Hex | Usage |
| --- | --- | --- | --- |
| background | Midnight Blue | `#112240` | primary page canvas — deep, focused canvas for developer/technical context |
| surface | Deep Navy | `#1a365d` | cards, panels, secondary surfaces — adds depth and hierarchy |
| accent | Sky Blue | `#7bb5e3` | brand accent — CTAs, logo center, highlights, interactive elements |
| foreground | Light Text | `#e8f1fb` | body text and headings on dark backgrounds |
| muted | Muted Blue | `#8ba3c7` | secondary text, metadata, placeholders on dark surfaces |
| border | Navy Border | `#2d4a6f` | rules, dividers, input borders on dark surfaces |
| accent-secondary | Steel Blue | `#5a9bd5` | secondary accent — hover states, secondary CTAs, supporting highlights |

## Typography
- **Display:** Inter — weights 400, 600, 700 — fallbacks: system-ui, -apple-system, Segoe UI, Helvetica Neue, Arial, sans-serif
- **Body:** Inter — weights 400, 500 — fallbacks: system-ui, -apple-system, Segoe UI, Helvetica Neue, Arial, sans-serif

## Voice & Tone

- **Adjectives:** Technical, Professional, Precise
- **Tone:** Speaks as an expert practitioner. Uses facts and clear logic over hype. Helpful and approachable while maintaining technical excellence standards.

### Messaging pillars
- Precision — Every line of code and design choice is intentional
- Technical Excellence — We don't just build; we optimize and master
- Reliability — A brand that stands for stability and trust

### Vocabulary
- **Use:** Implementation, Architecture, Efficiency, Scalable, Precision, Systems, Performance, Optimization
- **Avoid:** Synergy, Game-changing, Revolutionary, Disruptive, Best-in-class, Cutting-edge, World-class

## Imagery

- **Style:** Clean, line-based iconography; high-quality technical setups; abstract data visualizations
- **Subjects:** Code editor screenshots, Terminal/CLI output, Architecture diagrams, Technical documentation, Abstract geometric patterns
- **Treatment:** Professional and focused. Technical authenticity over polish. Real code and real outputs preferred over idealized illustrations.
- **Avoid:** Stock photos of people pointing at screens, Generic business imagery, Excessive use of gradients or decorative elements

## Layout

- **Radius:** 6px
- **Border weight:** 1px
- **Spacing:** 8px baseline grid

### Posture rules
- Deep midnight blue primary background creates focused, professional canvas
- Cards use Deep Navy (#1A365D) surfaces with subtle border separation
- Sky Blue accent used sparingly for CTAs, highlights, and interactive elements
- Component kit covers: Button, Card, Form Fields, Navigation, Code Block, Feature Grid, Testimonial, CTA Section
